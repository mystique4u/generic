# generic-cleanliness

Follow [docs/cleanliness-and-organization.md](../../../docs/cleanliness-and-organization.md).

## Checklist

- [ ] No unexpected top-level files (see `scripts/root-allowlist.txt`)
- [ ] Unused placeholders pruned or explained
- [ ] No dead code / commented-out blocks
- [ ] `docs/INDEX.md` updated if docs changed
- [ ] CHANGELOG updated if behaviour changed
- [ ] Secrets not committed
- [ ] English only

Leave the tree cleaner than you found it.
